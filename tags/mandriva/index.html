Cliente IRC Online para acessar o #mandriva-br

A Comunidade Mandriva Brasil conta com um canal de irc na rede frenode.net, o #mandriva-br. Nesse canal é possível bater um papo, falar sobre o Mandriva Linux ou mesmo buscar ajuda para solução de problemas com o uso ou configuração da distribuição. Para acesso ao canal você precisa de um cliente irc, para o desktop contamos com vários clientes como por exemplo o Xchat.
